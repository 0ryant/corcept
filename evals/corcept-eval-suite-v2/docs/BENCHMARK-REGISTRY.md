# Benchmark Registry

The eval suite exposes the benchmark registry through:

```bash
python -m corcept_eval list-benchmarks
```

Priorities are not prestige rankings. They are usefulness rankings for CORCEPT's claim:

1. skill marginal utility
2. repository issue correctness
3. terminal/runtime reliability
4. reasoning/correctness regression controls
5. policy/tool-use controls
6. long-horizon/research controls
